# `to-do`

Manage a to-do list using a simple command line tool

## Installation

```
go install
```

## Commands

Add a task to your list
```
to-do add
```

Complete a task
```
to-do complete
```

Show your to-do list
```
to-do show
```

`to-do` stores your current list in your home directory in a file called `to-do.json`.
